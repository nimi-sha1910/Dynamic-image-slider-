Dynamic Image Slider

🔗 Live Demo:
https://nimi-sha1910.github.io/Dynamic-image-slider-/

Overview

The Dynamic Image Slider is a front-end web project developed using HTML, CSS, and JavaScript. It displays multiple images in a sliding format with smooth transitions. The slider can automatically change images and also allows users to navigate manually using controls.

Features

Automatic image sliding,
Smooth transition effects,
Manual navigation controls,
Responsive design,
Clean and simple interface

Technologies Used

HTML,
CSS,
JavaScript

Project Structure

*index.html – Structure of the webpage
*style.css – Styling and layout
*script.js – JavaScript logic for the slider

Purpose of the Project

The purpose of this project is to build an interactive image slider and understand how dynamic UI elements work in front-end web development.
